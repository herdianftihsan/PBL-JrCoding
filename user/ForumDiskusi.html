Apa perbedaan let, var, dan const pada JavaScript?

Pertanyaan
Andika
2 Jam Lalu

var memiliki function scope dan bisa di-*redeclare*. let memiliki block scope dan tidak bisa redeclare. const sama seperti let tetapi nilainya tidak bisa diubah.

Catherine
5 Jam Lalu

var sering menimbulkan masalah karena hoisting. let lebih aman karena mengikuti block scope. const cocok untuk nilai tetap seperti API Key.

Rudi
1 Hari Lalu

var digunakan pada JavaScript lama. let dan const adalah standar modern sejak ES6. Gunakan let untuk variabel berubah, const untuk tetap.
